Rick And Morty Fan Site

HTML CSS JavaScript

Um projeto criativo inspirado no universo de Rick and Morty, desenvolvido com foco em prática e evolução no desenvolvimento web. O site conta com uso de HTML, CSS e JavaScript, trazendo uma interface interativa com imagens animadas e elementos visuais que remetem ao estilo único da série. Além do visual chamativo, o projeto demonstra domínio em estruturação de páginas, estilização e manipulação dinâmica de elementos.

Desenvolvido por: Thiago & Isabella
